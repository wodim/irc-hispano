\subsection{Detailed Description}
\begin{Desc}
\item[For internal use only.]
This file contains the implementation of the sparse matrix freelist and its support functions. The freelist is intended to avoid memory fragmentation due to the large number of hash entry structures that must be allocated by the sparse matrix system.\end{Desc}
